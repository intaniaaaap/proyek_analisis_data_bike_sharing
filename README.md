# proyek_analisis_data_bike_sharing

## Deskripsi
Proyek ini bertujuan untuk menganalisis pola penggunaan layanan bike sharing berdasarkan data historis penyewaan sepeda. Analisis difokuskan pada dua aspek utama, yaitu perbedaan pola penyewaan antara pengguna casual dan registered pada jam sibuk (peak hours) di hari kerja selama periode 2011–2012, serta identifikasi waktu terjadinya lonjakan penyewaan tertinggi.Selain itu, proyek ini juga mengevaluasi pengaruh faktor eksternal seperti musim dan kondisi cuaca terhadap fluktuasi jumlah penyewaan sepeda harian pada hari kerja selama tahun 2012. Analisis ini bertujuan untuk memahami sejauh mana kondisi lingkungan memengaruhi peningkatan maupun penurunan permintaan. Hasil analisis disajikan dalam bentuk dashboard interaktif menggunakan Streamlit untuk memudahkan eksplorasi data dan interpretasi insight yang dihasilkan.

## Setup Environment - Shell/Terminal
- mkdir proyek_analisis_data
- cd proyek_analisis_data
- pipenv install
- pipenv shell
- pip install -r requirements.txt

- Python

- GitHub
